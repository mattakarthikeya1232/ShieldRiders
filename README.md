# 🛡️ ShieldRider – DEVTrails Hackathon 2026 Submission

## 🏍️ Problem Statement

India has over **11 million gig delivery workers** (Zomato, Swiggy), but:

* ❌ No income protection
* ❌ No insurance for daily risks
* ❌ Zero earnings during rain, pollution, or curfews

Traditional insurance is:

* Slow (weeks for claims)
* Complex (forms, agents)
* High rejection rate (40–60%)

👉 Gig workers need **instant and automatic income protection**.

---

## 💡 Our Solution

**ShieldRider** is an **AI-powered parametric insurance system** that:

* Detects real-world events automatically (Rain, AQI, Curfew)
* Verifies rider location using GPS
* Runs AI-based fraud detection
* Sends **instant UPI payouts (<60 seconds)**
* Notifies users via WhatsApp

✔ No claim forms
✔ No manual process
✔ Fully automated

---

## ⚙️ Workflow

Trigger Detected → GPS Verified → Fraud Check → UPI Payout → Notification
